## How we coordinate
- **Inbox/outbox:** we message each other by dropping notes in a peer's `inbox/`.
- **Shared memory:** facts worth keeping go to collective memory; private notes
  stay in our own `memory/`.
- **Optional mesh + room:** for multi-host cohorts, agents reach each other over a
  private overlay network and a shared chat room. See `docs/mesh-and-comms.md`.
